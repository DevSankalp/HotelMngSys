# 🏨 Hotel Management System (ASP.NET Core Razor Pages)

A web-based **Hotel Management System** built using **ASP.NET Core Razor Pages**.  
This project helps hotel staff manage reservations, rooms, customers, and bookings efficiently through a simple and user-friendly interface.

---

## 📌 Features

- 🛏️ Room Management
  - Add, update, and delete rooms
  - Track room availability

- 📅 Booking Management
  - Create and manage reservations
  - Check-in / Check-out functionality

- 👤 Customer Management
  - Store guest details
  - View customer booking history

- 📊 Dashboard
  - Overview of hotel activity
  - Quick statistics for rooms and bookings

- 🔐 Authentication & Authorization
  - Secure login system
  - Role-based access (Admin / Staff)

---

## 🛠️ Technologies Used

- **ASP.NET Core**
- **Razor Pages**
- **Entity Framework Core**
- **SQL Server**
- **Bootstrap**
- **C#**
- **HTML / CSS / JavaScript**
